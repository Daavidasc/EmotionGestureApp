# EmotionGestureApp

## Sistema inteligente de detección de gestos y emociones en tiempo real.

EmotionGestureApp es una aplicación de Visión Artificial desarrollada en Python que procesa flujos de video en vivo para identificar gestos manuales y expresiones faciales. El sistema integra modelos de Deep Learning para ofrecer una respuesta rápida y precisa, ideal para aplicaciones interactivas o de accesibilidad.
## Características Principales

   Análisis en tiempo real: Detección fluida utilizando la cámara web.

   Procesamiento de Gestos: Seguimiento de landmarks (puntos clave) de la mano.

   Clasificación de Emociones: Inferencia de estados faciales mediante modelos optimizados.

   Arquitectura modular: Estructura desacoplada para facilitar el mantenimiento y la escalabilidad.

 ## Stack Tecnológico

   Lenguaje: Python 3.10

  Visión Artificial: OpenCV (Procesamiento), MediaPipe (Landmarks).

   Deep Learning: TensorFlow para la inferencia del modelo.

   Gestión de Datos: NumPy, Pandas (para la estructuración de los datos detectados).
